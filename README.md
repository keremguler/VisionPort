# ◈ VisionPort | Kişisel Finans ve Vizyon Portföyü

VisionPort, klasik Python/Tkinter kütüphanesinin sınırlarını zorlayarak modern, karanlık mod odaklı ve kullanıcı dostu bir finansal takip deneyimi sunar. 

> **Not:** Bu uygulama, macOS sistem mimarisi ve emoji render motoruyla tam uyumlu olacak şekilde optimize edilmiştir.

![Uygulama Ekran Kaydı](demo.gif)

## ✨ Öne Çıkan Özellikler
* **Modern UI/UX:** Özel animasyonlar, hover efektleri ve şık renk paleti.
* **Dinamik Borç Yönetimi:** Banka isimleri ve logoları ile görselleştirilmiş profesyonel borç takibi.
* **Başarı Serisi (Streak):** Finansal disiplininizi oyunlaştıran günlük takip sistemi.
* **Yapay Zeka Danışmanı (Simülasyon):** Mevcut finansal durumunuza göre strateji üreten analiz modülü.
* **İnteraktif Grafikler:** Borç dağılımınızı gösteren animasyonlu ve interaktif pasta grafiği.

## 🛠 Teknik Detaylar
* **Dil:** Python 3
* **Arayüz:** Tkinter (Custom UI Components)
* **Veri Yönetimi:** JSON tabanlı yerel depolama
* **Görsel İşleme:** Pillow (PIL)
* **Uyumluluk:** macOS
